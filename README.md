# ⚡ リアクションダッシュ

あか → **みどり** になったしゅんかんにタッチする、子供向けの反応速度ゲームです。かわいいマスコットと効果音つきで、5回の平均タイムからランク（にんじゃ・チーターなど）が決まります。

🎮 **あそぶ**: https://tommm87.github.io/tttttt87/

## あそびかた

1. 画面をタッチ（またはスペースキー）してスタート
2. あかいあいだは **まつ**
3. **みどり**になったら、できるだけはやくタッチ！
4. これを5回くりかえすと、へいきんタイムとランクが表示されます

早く押しすぎると「はやすぎ！🙈」になり、その回をやりなおします。

## 特徴

- 📱 マウス・タッチ・スペースキーに対応
- 🏆 ベスト記録はブラウザ（`localStorage`）に保存
- 🎨 依存ライブラリなし。SVGマスコットとCSSアニメーションのみ
- 🔊 Web Audio API による効果音

## 構成

`index.html` 1ファイルに HTML・CSS・JavaScript をまとめています。ビルド不要で、ブラウザで開くだけであそべます。
